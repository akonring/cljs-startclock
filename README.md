# cljs-startclock

A cljs app that is supposed to be a browser startclock.

## Usage

Current Usage
--
A this moment the startclock will beep every minute.

* Fire up a repl
* Step into cljs-startclock.core
* Evaluate 'run' to run the server on localhost/8080
